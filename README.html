Autor: Fábio Moura de Oliveira


Por que criei este programa???

Este programa foi criado com o objetivo de aumentar as chances de ganhar na loteria

Obs.: A utilização deste programa não irá garantir que você sempre ganhará na loteria

O que este programa faz???

Este programa realiza análise de jogos lotéricos da caixa e da intralot.

O programa está sendo implementando, ainda não está completo
